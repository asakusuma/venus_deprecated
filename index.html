VENUSJS

Simple, Unified JS Testing

Organize and run your entire JS testing system with a single tool, using Node.js. Integrates with Mocha, QUnit, Jasmine, PhantomJS, Selenium, and Casper.

Version 0.9.1

Automate your JS testing with Venus

Venus is bringing sexy back when it comes to JavaScript testing. By eliminating repetitive tasks and friction points, Venus lets you focus on writing tests instead of running them. Already have JavaScript testing in place? Perfect! Venus is designed to orchestrate and automate existing tools like Mocha and Casper. See our partners in crime for more information on integration with other toots.

Quick Start

  1. Install Node.js
  2. Install PhantomJS
  3. Run the following commands in Terminal
git clone https://github.com/venusjs/venus.git
cd venus && npm install
  1. Add venus/bin to your path
  2. Verify installation
venus run --test venus/examples/01-SimpleMochaTest/specs/Greeter.spec.js --phantom
